# Source and Sink Rules for FlowDroid

All files in this directory define source and sink rules for FlowDroid. 
These rules 
help identify potential data leaks by specifying where sensitive 
data can originate (sources) and where it can be sent (sinks).

Following are the files included in this directory:
- codoc_source_sinks.txt: Original from [https://github.com/JordanSamhi/CoDoC/tree/master](https://github.com/JordanSamhi/CoDoC/tree/master)
